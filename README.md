## 🛠️ Technologies

| Technologie | Usage |
|---|---|
| HTML5 / CSS3 | Structure sémantique, Flexbox/Grid, Responsive |
| jQuery 3.7 | Animations, accordéon, barres de compétences, scroll |
| React 18 | `<ProjectCard />`, `<ContactForm />`, `<ProjectsGrid />` |
| GitHub Pages | Hébergement gratuit & déploiement continu |

## ✏️ Personnalisation rapide

1. **Votre nom** : Cherchez `Votre Nom` dans `index.html` et remplacez
2. **Photo** : Remplacez le bloc `.profile-img-placeholder` par `<img src="photo.jpg" class="profile-img" />`
3. **Projets** : Modifiez `projectsData` dans `components.jsx`
4. **Compétences** : Modifiez les `data-width` des `.skill-fill` dans `index.html`
5. **Email/LinkedIn/GitHub** : Cherchez `username` et `votre@email.com`
